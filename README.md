# WebGPU COPC Viewer

to install and run:

```
1. clone the repo
```
```
2. npm install
```

### Note:

<b>```Renderer is made using WebGPU and WGSL and does out of core rendering so check for support of WebGPU to run this code in your browser```</b>


```
3. npm run dev
```
